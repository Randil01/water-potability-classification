# Water Potability Prediction using Machine Learning

## Project Overview

This project applies classical machine learning algorithms to predict whether water is safe for human consumption based on physicochemical properties. The goal is to compare multiple machine learning models and evaluate their performance on the same dataset.

## Problem Statement

Access to clean and safe drinking water is essential for public health. This project aims to develop machine learning models that can classify water samples as potable or non-potable using water quality indicators.

## Dataset

The dataset used in this project is the **Water Potability Dataset** available on Kaggle.

Dataset Link:
https://www.kaggle.com/datasets/adityakadiwal/water-potability

The dataset contains 3,276 water samples with 9 chemical and physical properties such as pH, hardness, solids, chloramines, sulfate, conductivity, organic carbon, trihalomethanes, and turbidity. The target variable `Potability` indicates whether the water is safe for drinking (1) or not (0).

## Machine Learning Algorithms Used

Four different machine learning algorithms are implemented and compared:

1. Logistic Regression
2. Decision Tree (CART)
3. Random Forest
4. Support Vector Machine (SVM)

Each group member is responsible for implementing and evaluating one algorithm.

## Data Preprocessing

The following preprocessing steps were applied:

* Handling missing values using mean imputation
* Feature scaling using StandardScaler
* Splitting dataset into training and testing sets (80/20)

## Evaluation Metrics

Model performance is evaluated using:

* Accuracy
* Precision
* Recall
* F1-score
* Confusion Matrix
